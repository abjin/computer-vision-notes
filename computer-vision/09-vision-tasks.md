# 다양한 비전 작업 종류

1. 분류 - 이동 불변성
  - 이미지 전체에 하나의 라벨을 부여
2. 분류 + 위치 추정 - 이동 가변성
  - 한 객체에 대해 클래스 라벨과 바운딩 박스를 예측
3. 객체 탐지 - 이동 가변성
  - 다수의 객체를 탐지하며, 각각의 바운딩 박스와 라벨을 가짐
4. 인스턴스 분할 - 이동 가변성
  - 같은 클래스의 객체라 할지라도 서로 구분하는 클래스 마스크

# 세그멘테이션 종류

| **구분**        | **시맨틱 분할 (Semantic Segmentation)** | **인스턴스 분할 (Instance Segmentation)** |
| ------------- | ---------------------------------- | ----------------------------------- |
| **출력 라벨**     | 픽셀당 클래스 ID                         | 픽셀당 클래스 ID + 인스턴스 ID                |
| **동일 클래스 객체** | 같은 라벨                              | 서로 다른 라벨 (인스턴스 구분)                  |
| **사용 사례 예시**  | 도로 장면 분석, 의료 영상                    | 자율 주행, 객체 카운팅(개수 세기)                |

# 이동 불변성

- 표준 CNN 에서의 이동 불변성
  - Conv 연산 -> 기본적으로 이동 등변성을 가짐, 하지만 깊어질수록 해상도가 감소하여 위치 정보 소실
  - 가중치 공유 -> 동일한 필터가 이미지의 어느 곳에서든 패턴을 감지함 (이동 불변성)
  - 풀링 -> 작은 이동에 대한 민감도를 줄임 (소규모 불변성)
  - 완전 연결 계층 + softmax: 거의 모든 위치 정보를 버림 (이동 불변성)

- 분류에서의 이동 불변성
  - 이동 불변성이 필요함

- 탐지에서의 이동 불변성
  - 이동 가변성이 필요함, 이동 불변성은 지역성 손실을 야기하여 픽셀 단위의 예측을 저해 할 수 있음

# 분할과 탐지의 핵심 요구 사항

- 탐지를 위한 핵심 요구사항
  - 공간 정보 보존: 정확한 위치 파악
    - 공간 정보: 이미지 내 특징들의 정확한 위치 정보
    - 모델이 객체가 어디에 위치하는지 알 수 있도록 함.
  - 지역성 보존: 구성 요소간의 관계 파악
    - 지역성: 이미지의 특정 영역 내의 특징들의 배치 정보
    - 객체 내부 구조를 포착하는 데 중요

- 분할을 위한 핵심 요구사항
  - 고수준 특징: 분류를 위한 시멘틱한 정보
  - 저수준 특징: 정밀한 경계를 위한 정보
